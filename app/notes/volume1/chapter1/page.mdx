# 第一章 基础

### 病毒的定义

病毒是一类极微小的感染物，仅能在其感染的细胞内繁殖：它们是其宿主细胞的专性寄生体。病毒在细胞内传播的媒介是一种被称为病毒颗粒的传染性粒子，这些病毒颗粒由一个保护性的蛋白质外壳包裹着DNA或RNA基因组。随着病毒颗粒进入宿主细胞并裂解，病毒基因组指导细胞系统合成病毒组分。在受感染的细胞中，新合成的组分重新自行组装，就形成了子代病毒颗粒。

- 病毒是专性细胞内寄生物，其增值的各个环节都依赖于宿主细胞。
- 病毒可以感染所有生物，包括昆虫、植物和细菌。
- 病毒不仅仅有致病的危害，它们也可以是有益的。病毒有助于维持生态平衡，能使我们的免疫系统维持活跃和警觉，还可以用作“分子手电”来探明细胞过程。
- 病毒可以根据它们的外观、感染的宿主或核酸基因组的性质来描述。
- 所有病毒都必须产生可被细胞核糖体翻译的mRNA。巴尔的摩分类使得根据mRNA产生途径可确定含RNA或DNA基因组的各种病毒间的关系。
- 所有病毒的增殖共用一套通用程序。

_人体平均含有10<sup>13</sup>个细胞，但体内细菌数量几乎与之相等，而病毒颗粒数量则多出100倍。_\
_如今地球上大约有10<sup>16</sup>个人类免疫缺陷病毒1型（HIV-1）基因组，因此，在某个地方很可能存在着某种HIV-1基因组，它对我们现在拥有或将来可能拥有的每一种抗病毒药物都具有耐药性。_

### 病毒“是”我们

我们体内的每个细胞都含有病毒DNA。人类内源性反转录病毒及其成分约占人类基因组的8%，不过它们大多数是不活跃的，是我们数百万年进化过程中生殖细胞感染所残留下来的。

### 病毒能跨越物种屏障

#### 科赫假设

验证特定疾病的因素（病原体）的原则

- 该生物体必须与疾病及其特征性病变有规律性的关联；
- 该生物体必须能够从患病宿主中分离出来并可以在培养基中生长；
- 当该生物体的纯培养物引入健康易感宿主时，该疾病必须能够重现；
- 同样的生物体必须能够从实验感染的宿主中重新分离获得。

### 病毒的发现

1892年，俄国科学家迪米特里·伊万诺夫斯基，烟草花叶病的病原体。

#### 噬菌体的发现

弗雷德里克·特沃特和费利克斯·德·埃雷尔分别于1917年和1917年发现了细菌也可以成为病毒的宿主。德·埃雷尔将这种病毒命名为噬菌体。

#### 溶原现象

某些细菌菌株在培养一段时间后可以自发裂解并产生噬菌体，这种菌株被称为溶原性菌株，这种现象被称为溶原现象。

#### 类病毒 朊病毒

病毒不是最简单的生物活性物质：类病毒，对多种具有重要经济价值的植物具有感染性，仅有单个小分子的非编码核糖核酸组成；而被称为朊病毒的物质会导致人类和动物的神经系统疾病，被认为是单个蛋白质分子的聚合物。

```
病毒是生命体吗？

病毒可以被视为以两个（生命）阶段存在的微生物：
一个是无生命阶段，及病原体；
另一个是被感染细胞中的增殖阶段。
```
